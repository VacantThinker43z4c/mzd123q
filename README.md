## 简介

桌面版脑图是基于百度脑图的本地化版本，帮助你在没有互联网环境的情况下，依然可以使用脑图工具。

## 功能
- 包含百度脑图的基本功能
- 本地km文件的操作
- [提供百度脑图文件的下载脚本](doc/Help.md)
